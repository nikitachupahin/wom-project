# WOM - Test Task: Mobile Header & Text Block

## Встановлення та запуск

1. Скопіювати репозиторій:

```bash
https://github.com/nikitachupahin/wom-project.git
```

2. Перейдіть до каталогу проекту:

```Bash
cd wom-project
```

3. Встановити пакети NPM:

```Bash
npm install
```

4. Запустити проект:

```Bash
npm run dev
```

Додаток буде доступний за адресою [http://localhost:5173/](http://localhost:5173/).
